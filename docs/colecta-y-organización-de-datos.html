5 Colecta y organización de datos

5.1 Exportar plantilla de evaluación

Vamos a crear el diseño experimental de nuestra idea de investigación haciendo uso del paquete agricolae (de Mendiburu 2019) que permité hacer el diseño de varios modelos para experimentos agricolas.

En base a nuestro plan experimental tenemos la siguiente información

Table 5.1: Información básica para el experimento
Variables Abbreviation Units/Class Description
Identificacion ID Factor Unique identification for each EU
Repetition RPT Factor 4 repetitions
Imbibition time IBT Factor 13 imbibition time each 2 hours

Con la información de la tabla (tab:dse) diseñaremos nuesto libro de campo y la distribución del experimento en el área de trabajo.

library(agricolae)

imb <- seq(from = 0, to = 24, by = 2)
rep <- 4

fbd <- design.rcbd(trt = imb,
                  r = rep,
                  seed = 20, 
                  serie = 2)

fb <- fbd$book %>% 
  rename(
    ID = plots,
    Block = block,
    IBTH = imb) %>% 
  mutate(IBT = paste0(IBTH, "hrs")) %>% 
  mutate(Block = recode(Block,
                        "1" = "I",
                        "2" = "II",
                        "3" = "III",
                        "4" = "IV"))

fds <- zigzag(fbd)
fds <- matrix(fds[,1],byrow=TRUE,ncol=13) %>% 
  as.data.frame()

Libro de campo para colecta de información, el cual sera usado para las etiquetar las unidades experimentales y el libro de colecta de datos para evaluar los experimentos con nuestra tablet o celular.

La distribución de nuestro experimento en el campo o invernadero seria la siguiente.

5.2 Etiquetas de unidades experimentales

El etiquetado de las unidades experimentales es una actividad que siempre se deja de lado, y muchas veces por el trabajo que genera. Pues piensalo 2 veces antes de dejar esta actividad para el final, un etiquetado adecuado, facilitara la evaluación y toma de datos en todas las instancias del proyecto.

5.4 Evaluacion de experimentos

Independientemente del tipo de diseño que hayas elegido, este será utilizado para analizar los datos, la informacion sera tomada e inserida al libro de campo para cada unidad experimental.

Los materiales que necesitamos son los siguientes:

  1. Libro de campo o fieldbook (fb)
  2. Libro de colecta de datos impreso
  3. Tablet o celular
  4. ScanPet app https://play.google.com/store/apps/details?id=com.maiko.xscanpet&hl=es-419
